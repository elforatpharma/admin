# نظام كوبونات الخصم - الفرات فارما

## ✅ ما تم إضافته

### 1. في لوحة التحكم (`admin.html`)

#### أ. قائمة جانبية جديدة
- تمت إضافة رابط "الكوبونات" في القائمة الجانبية مع أيقونة `discount`

#### ب. قسم إدارة الكوبونات
- جدول عرض الكوبونات مع الأعمدة التالية:
  - كود الكوبون
  - نسبة الخصم
  - تاريخ الانتهاء
  - الحالة (نشط/متوقف/منتهي)
  - إجراءات (تفعيل/إيقاف/حذف)

#### ج. مودال إضافة كوبون
- حقل كود الكوبون (يُحول تلقائياً لأحرف كبيرة)
- نسبة الخصم (1-99%)
- تاريخ الانتهاء (منتقي تواريخ)
- وصف اختياري للكوبون

#### د. الوظائف البرمجية
```javascript
- showCouponsSection()       // التنقل لقسم الكوبونات
- fetchCoupons()             // جلب وعرض جميع الكوبونات
- toggleCoupon(id, state)    // تفعيل/إيقاف كوبون
- deleteCoupon(id)           // حذف كوبون
- openCouponModal()          // فتح مودال الإضافة
- closeCouponModal()         // إغلاق المودال
- initCoupons()              // تهيئة النظام ومعالجة النموذج
```

### 2. ملف قاعدة البيانات (`coupons_schema.sql`)

#### جدول `coupons`
| العمود | النوع | الوصف |
|--------|-------|--------|
| id | UUID | المعرف الفريد |
| code | TEXT | كود الكوبون (فريد) |
| discount_percentage | INTEGER | نسبة الخصم (1-99) |
| expiry_date | DATE | تاريخ الانتهاء |
| description | TEXT | وصف الكوبون |
| is_active | BOOLEAN | الحالة (نشط/متوقف) |
| created_at | TIMESTAMP | تاريخ الإنشاء |
| updated_at | TIMESTAMP | تاريخ آخر تحديث |

#### فهرسة متقدمة
- فهرس على `code` للبحث السريع
- فهرس مركب على `is_active` و `expiry_date`

#### سياسات الأمان (RLS)
- قراءة/كتابة/تحديث/حذف للمستخدمين المصادقين فقط

## 🎯 كيفية الاستخدام

### إضافة كوبون جديد

1. اضغط على زر "الكوبونات" في القائمة الجانبية
2. اضغط "إضافة كوبون"
3. املأ البيانات:
   - **كود الكوبون**: مثلاً `WELCOME20`
   - **نسبة الخصم**: مثلاً `20`
   - **تاريخ الانتهاء**: اختر التاريخ
   - **الوصف**: (اختياري) مثلاً "خصم ترحيبي"
4. اضغط "حفظ الكوبون"

### إدارة الكوبونات الموجودة

- **تفعيل/إيقاف**: اضغط زر "تفعيل" أو "إيقاف" حسب الحالة
- **حذف**: اضغط زر "حذف" وأكد العملية
- **الحالة التلقائية**: الكوبون يُعتبر "منتهي" تلقائياً بعد تاريخ الانتهاء

## 🔗 التكامل مع صفحة الدفع (Checkout)

لاستخدام الكوبونات في متجر العملاء، أضف الكود التالي في صفحة الدفع:

```javascript
async function applyCoupon(couponCode) {
  const { data, error } = await _supabase
    .from('coupons')
    .select('*')
    .eq('code', couponCode.toUpperCase().trim())
    .eq('is_active', true)
    .gte('expiry_date', new Date().toISOString().split('T')[0])
    .single();

  if (error || !data) {
    return { success: false, message: 'كود الكوبون غير صالح' };
  }

  // حساب الخصم
  const discountAmount = (total * data.discount_percentage) / 100;
  const finalTotal = total - discountAmount;

  return {
    success: true,
    discount: discountAmount,
    finalTotal: finalTotal,
    couponCode: data.code
  };
}
```

## 📝 ملاحظات هامة

1. **الكود فريد**: لا يمكن إضافة كوبونين بنفس الكود
2. **الأحرف الكبيرة**: يتم تحويل الكود تلقائياً لأحرف كبيرة
3. **التاريخ**: الكوبون ينتهي في الساعة 00:00 من تاريخ الانتهاء
4. **الأمان**: فقط المستخدمين المسجلين يمكنهم إدارة الكوبونات
